Sistema de gestion de inventario

El objetivo de este proyecto es crear un sistema de gestión de inventario que permita a los usuarios registrar productos, almacenar información sobre los productos y realizar búsquedas en los productos registrados.

El sistema debe tener una interfaz de usuario intuitiva y fácil de usar, y debe ser capaz de realizar búsquedas en los productos registrados.

Además, el sistema debe ser capaz de generar informes detallados sobre los productos registrados, incluyendo la cantidad de productos registrados, la cantidad de productos en stock y la cantidad de productos en stock.

Sistema de gestión de clientes

El objetivo de este proyecto es crear un sistema de gestión de clientes que permita a los usuarios registrar información sobre sus clientes, almacenar información sobre sus clientes y realizar búsquedas en la información registrada de sus clientes.

El sistema debe tener una interfaz de usuario intuitiva y fácil de usar, y debe ser capaz de realizar búsquedas en la información registrada de sus clientes.

Además, el sistema debe ser capaz de generar informes detallados sobre la información registrada de sus clientes, incluyendo la cantidad de clientes registrados, la cantidad de clientes con una cuenta activa y la cantidad de clientes con una cuenta inactiva.
